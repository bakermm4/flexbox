flex-box

A way to distribute elements easily and evenly!

This webpage is designed to demonstrate how the flex-box property works, and how easy it is to use. Forget using inline-block and worrying about setting perfect margins. Now we can use flex-box to do it all.

These styles are applied to parent elements

Justify-Content

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around

Using flex-box is very simple. In your normal css you can use it to replace inline-block to make your life much easier. All you have to do is set the parent for your elements to display: inline-flex;. After that, you can set the justify-content property to whatever you want. The individual properties are demonstrated above.

Align-Items

  • flex-start
  • flex-end
  • center
  • baseline

The align-items property is very similiar to the vertical-align property, but it is much easier to use because you only have to apply the style once to the parent element. The individual properties are shown above. Note that the baseline property aligns the elements along the baseline of the text. There is a property not shown here called stretch. It stretches the elements to fit vertically unless a height and width are specified.
